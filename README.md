# Lightseek theme

## Configuration

The theme options are loaded from package.json. The entry lightseek_options includes the following variables:

- scripts - decides whether scripts.js should be loaded in the frontend.
- widgets - enables theme widgets.
- webfonts - array of google font families to be loaded in WebFont script. Empty array prevents the WebFont script from being enqueued. Example: `["Montserrat:400,400i,500,500i", "Cardo:400,400i,700"]`.
